关于华强北手表 | | | | | | | | | | |
**** | | | | | | | | | | |
首先，华强北手表这玩意本质上就是个安卓系统的小手机，价格基本在200~400，截至目前最大内存组合只有6+128，处理器一般为展讯8541e/w377e/w527和翱捷ASR8601/ASR8601C，性能一般没法和手机比，其次它的定位学生或者没手机时替��使用，切勿以手机标准要求手表，电池和美观不可兼得，因为体积限制所以散热一般般,发热还是有点大的，电池也大不到哪去，最后强调性能受制于成本基本就只有这样，防水相当于没有，请注意避开水或者液体，几百块的东西做成这样已经很不错了，毕竟这价格真的不能要求有什么特别好的，要求请不要太过于坎坷了 | | | | | | | | | | |
**** | | | | | | | | | | |
**** | | | | | | | | | | |
1.关于系统：首先，将这个列在第一个，并非没有原因，手表的系统与使用舒适度还是有很大的关系，花了几百买的手表，可天天一堆BUG，还时不时卡死，这个我相信大家是不愿意的，其次是功能上，相信方便丰富简洁的内置功能对于大家来说，也是很重要的，最后就是，希望大家看完表格能买到自己心仪的表，系统好用程度排序如下：和顺成MIMU UI(及1.0到4.0）>小糯米NOEMIE OS>小��米系统>微卫士>三基同创（ONMI AOS）/子腾 （最后这俩和HAMI OS我就不排序了，可能会有些争议） | | | | | | | | | | |
**** | | | | | | | | | | |
2.关于屏幕：华强北手表屏幕要看的不只是分辨率，主要看它是OLED还是LCD，触控点（有的因为触控点只有2点，开了压��灭屏很容易黑屏）还有屏幕像素密集度（ppi），当然分辨率也是必不可少的，据目前我体验过的华强北手表，可以列出以下排行：640x480 286ppi>412x502（410x502）272ppi>640x480 160ppi>480x480 160ppi>410x502 129ppi>368x448 110ppi | | | | | | | | | | |
**** | | | | | | | | | | |
3.关于处理器：处理器主要是展讯w527/8541e/w377e/8521e，以及虎贲T310，翱捷ASR8601/ASR8601C，联发科MT6761/MT6762/MT6739/MT6737M/MT6580，虽然有的处理器表中的手表没有用，但是有表确实在用，手表基本性能都不高，基于我已知，可以按如下排序：展讯W917>虎贲T310>展讯W527>联发科MT6762>联发科MT6761>翱捷ASR8601C（LPDDR4）>展讯8541E>展讯w377e>翱捷ASR8601C>翱捷ASR8601>联发科MT6739>联发科MT6737>联发科MT6580>展讯8521e（不代表全部，仅代表个人所知，如有争议，可提出修改） | | | | | | | | | | |
**** | | | | | | | | | | |
4.关于RAM：不要以为这个东西不起眼，这个东西对性能还是很大的，小苏米yu系列的翱捷ASR8601C就是最好的例子，一般展讯w527用的是LDDPR3，后续可���会上LPDDR4，展讯8541e/w377e用的是LPDDR2/3，翱捷ASR8601/翱捷ASR8601C是LPDDR3/4 | | | | | | | | | | |
5.关于电池：华强北手表电池质量并不统一，部分手表电池质量很差，与容量出入巨大，电池寿命掉的很快，目前华强北手表最大的电池容量是1400毫安，最好的电池是m9u，m53u，m12m等上面用��1000（960）及其他行胜工厂的高容高压电池，其次是卫派2025年后出的表，整治过后寿命提升很大，对于电池我就不做排序了，除了以上列出来的，基本就是看容量了，如有不符请提出修改 | | | | | | | | | | |
